## [简介](https://github.com/SoCXin/XL2409/wiki)

[XL2409](http://www.xinlinggo.com/pd.jsp?id=36#_jcp=3_10) 芯片是一款高性能低功耗的SOC集成无线收发芯片，集成M0核MCU,工作在2.400~2.483GHz世界通用ISM频段。该芯片集成了射频接收器、射频发射器、频率综合器、GFSK调制器、GFSK解调器等功能模块，并且支持对多线网和带ACK的通信模式。发射输出功率、工作频道以及通信数据率均可配置。

芯片集成多路I2C、SPI、USART等通讯外设，1路12bitADC, 5个16bit定时器，以及2路比较器。

### 关键参数

* 48 MHz [Cortex-M0](https://www.soc.xin/Cortex-M0)
* 8KB SRAM + 64KB Flash
* USART + SPI + I2C
* 12-bit ADC (9ch)
* TX +8dBm
